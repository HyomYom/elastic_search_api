# Elastic_Search_API

# 프로젝트 소개

## 개요
Node.js 기반 Elastic을 활용한 검색 엔진 서비스

[요구사항]
1. 기능 검색(category, title, reporter) 검색
2. 필터 기능(정확도순, 최신순)
3. 검색 기간 설정
4. 하이라이트 기능(검색어 연관 핵심 단어 자동 하이라이트)


# 개발 환경
- Node 18.14.1 / npm 9.3.1
- IDE : Visual Studio Code
- Database : Elastic 7.10.2
- Tools : Ejs, NiFi, Logstash, Kibana

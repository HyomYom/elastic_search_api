# Elastic_Search_API

#프로젝트 소개

## 1.개요
Node.js 기반 ElasticSearch를 활용한 검색 서비스

## 2.개발 배경



[요구사항]
1. 기본적인 주문 서비스 기능(장바구니, 결제)
2. 고객관리
3. 주문 외 시간 주문 방지
4. 픽업 예상시간 설정
5. 품목(메뉴) 조절 설정
6. 매장관리(주문 설정, 매장 등록/삭제)


# 개발 환경
- Node 18.14.1 / npm 9.3.1
- IDE : Visual Studio Code
- Database : Elastic 7.10.2
- Tools : Ejs, Logstash, Kibana
